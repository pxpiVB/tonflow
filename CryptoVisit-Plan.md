# CryptoVisit — Project Plan

## Що це таке

Treasure hunt у реальному світі + крипто.
У кафе захований QR-код. Знайшов → відсканував → отримав CVT токени.
Після TGE (лістинг на біржі) токени можна продати або тримати.

**CVT — обмежений токен: лише 100 000 000 штук, більше ніколи не буде.**

---

## Як це працює (потік)

```
КАФЕ                             КЛІЄНТ
─────                            ───────
Реєструється (1 раз)        →    Приходить у кафе
Отримує унікальний QR            Шукає прихований QR-код
Ховає його в закладі             Знаходить → сканує телефоном
                            ←    Підключає гаманець
                            ←    Отримує 10 CVT
                                 Копить токени
                                 Після TGE — продає або тримає
```

## Компоненти проекту

### 1. Smart Contract (Base blockchain)
- ERC-20 токен CVT ✅ НАПИСАНО
- Реєстрація кафе
- Видача токенів при відвідуванні
- Редемпція токенів на знижки

### 2. QR-код сторінка (веб-додаток)
- Клієнт сканує QR → відкривається сторінка
- Підключає гаманець (або входить через email)
- Натискає "Отримати токени" → транзакція на контракт
- Показує баланс CVT

### 3. Панель для кафе (адмін)
- Кафе бачить свою статистику (скільки відвідувань, активних клієнтів)
- Кнопка "Списати токени" при оплаті
- Генерація QR-коду

### 4. Лендінг сайт
- Для верифікації на talent.app
- Пояснює концепцію для кафе та клієнтів
- Форма для реєстрації кафе

## Токеноміка

| Параметр | Значення |
|----------|---------|
| Назва | CryptoVisit Token (CVT) |
| Мережа | Base (Ethereum L2) |
| Максимальна кількість | 100 000 000 CVT (ніколи більше) |
| За 1 відвідування | 10 CVT |
| Кулдаун | 4 години (те саме кафе) |
| TGE | Uniswap v3 або Aerodrome на Base |

**Як CVT отримує цінність:**
- Обмежена кількість — 100M назавжди
- Чим більше кафе → більше людей → більший попит
- До TGE токени накопичуються (FOMO)
- На TGE власники вирішують: продати або тримати
